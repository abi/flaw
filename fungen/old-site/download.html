There is currently only one FunGEn release:

  • FunGEn-0.1 - The very first release. Win32 platform users must download the FunGEn Win32 version (zip file, 75 Kb). For other users, please download the other version (tar.gz file, 66 Kb). The only difference between these two versions is the sound support, avaiable only in the win32 version (hope this will be changed soon!).

In order to use FunGEn, you'll need also to have the following programs installed:

  • The Glasgow Haskell Compiler - version 5.0 or higher. You can download GHC from its official site.
  • HOpenGL - please check for HOpenGL-1.01 in its official site.

If you just would like to see some examples, help yourself below (only win32 binaries are avaiable by now):

  • Pong (424 Kb) by Andre W B Furtado - A simple version of the game Pong. Use the arrow keys to move a bar and try to hit the ball. Get source code here.

  • Worms (434 Kb) by Andre W B Furtado - Move the snake around and eat the food along its way. Do not hit the wall or yourself. Get source code here.

  • Your game here...! - if you wish your game to be placed here, send me your source code (please zip or tar the bmp, wav and other files).

Perhaps some DLLs will be required. You can find them in my HOpenGL examples site.
